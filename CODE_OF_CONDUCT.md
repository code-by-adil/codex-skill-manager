# Code of Conduct

Codex Skill Manager aims to be a practical, welcoming open-source project.

Participants are expected to:

- Be respectful and constructive.
- Assume good intent while still being clear about problems.
- Keep feedback focused on the work.
- Avoid harassment, insults, or discriminatory language.
- Respect privacy when discussing project paths, skill contents, logs, or screenshots.

Maintainers may edit, hide, or remove comments and issues that make the project harder for others to participate in safely and productively.
